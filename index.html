Un nœud dans l'arborescence DOM est l'intersection de deux branches contenant des données. Les nœuds peuvent représenter des éléments HTML, du texte, des attributs, etc. Le nœud racine est le nœud le plus haut de l'arborescence. L'illustration montre une représentation d'un DOM contenant différents types de nœuds.
